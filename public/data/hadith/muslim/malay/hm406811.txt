40|6811|  Bab: Penerangan tentang neraka dan kepanasan panasnya dan siksaannya.  Abil Huraira meriwayatkan bahawa Rasulullah saw bersabda: Api yang dibakar oleh anak Adam hanyalah satu per tujuh puluh bahagian dari api Neraka. Para sahabat berkata: Demi Allah, api biasa pun sudah cukup (membakar manusia). Kemudian dia berkata: Ia adalah enam puluh sembilan bahagian lebih daripada (panas) api di dunia ini setiap satu daripadanya adalah sama dengan panasnya.  INTIM
