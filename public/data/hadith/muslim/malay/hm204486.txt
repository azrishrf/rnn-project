Perawi berkata: Saya telah bersumpah bahawa saya akan bercakap dengannya tentang perkara itu. Saya mendiamkan diri sehingga keesokan paginya, masih saya tidak bercakap dengannya, dan saya merasakan seolah-olah saya membawa gunung di sebelah kanan saya. Akhirnya saya datang kepadanya dan memasuki apartmennya. (Melihat saya) dia mula bertanya kepada saya tentang keadaan orang ramai, dan saya memberitahunya (tentang mereka).  Kemudian saya berkata kepadanya: Saya mendengar sesuatu daripada orang ramai dan bersumpah bahawa saya akan menyampaikannya kepada anda. Mereka menganggap bahawa anda tidak akan mencalonkan pengganti. Jika kembali kepada kamu seorang pengembala unta dan kambing yang telah kamu jadikan dengan meninggalkan ternakan itu, nescaya kamu akan menyangka bahawa binatang-binatang itu sesat. Untuk menjaga rakyat adalah lebih serius dan serius. (Khalifah yang hampir mati) terharu dengan kata-kata saya. Dia menundukkan kepalanya dalam suasana termenung untuk beberapa lama dan mengangkatnya kepada saya dan berkata: Tuhan pasti akan melindungi agama-Nya. Jika saya tidak mencalonkan penggantinya (saya ada dahulu sebelum saya), kerana Rasulullah saw tidak mencalonkan penggantinya. Dan jika saya mencalonkan seorang (saya mempunyai contoh), kerana Abu Bakar telah mencalonkan. Perawi (Ibn Umar) berkata: Demi Allah, apabila dia menyebut Rasulullah saw dan Abu Bakar, saya (segera) faham bahawa dia tidak akan meletakkan sesiapa pun setanding dengan Rasulullah ( semoga sejahtera ke atasnya) dan tidak akan mencalonkan sesiapa pun.
