Suatu ketika saya ingin melihat mimpi dan menceritakannya kepada Nabi (ï·º), saya masih muda, belum berkahwin, dan pernah tidur di Masjid semasa hayat Nabi. Saya bermimpi bahawa dua malaikat membawa saya dan pergi bersama saya ke arah Neraka (Neraka) yang kelihatan seperti telaga dengan dinding di dalam dibina, dan mempunyai dua dinding sisi seperti telaga. Di sana saya melihat beberapa orang di dalamnya yang saya kenali. Aku mula berkata, "Aku berlindung dengan Allah dari api neraka, aku berlindung dengan Allah dari api neraka."
